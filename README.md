# Banking System 🏦

A console-based banking application written in Python.

## Overview

Banking System is a Python project that simulates basic banking operations.
The project focuses on object-oriented programming, file persistence,
input validation, and modular code organization.

## Features

### Account System
- Create bank accounts
- Generate account numbers
- PIN authentication
- Change PIN
- Delete accounts
- View account information

### Banking Operations
- Deposit money
- Withdraw money
- Transfer money between accounts
- Transaction history

### Security
- PIN validation
- Failed login tracking
- Account locking after failed attempts

### Data Storage
- JSON-based save system
- Accounts persist after restarting

## Technologies

- Python 3
- JSON
- Object-Oriented Programming
